Estilos CSS: las clases

Dentro de la hoja se pueden aplicar distintos estilos según la clase de etiqueta. La clase se puede especificar con el atributo class.

En el fichero CSS una clase se define con un nombre precedido por un punto como por ejemplo: .párrafo_especial { propiedad1:valor1; propiedad2: valor2 }
